Mando Chest Display is an app that simulates the chest display of Boba Fett, character of Star Wars films.

The app allows the user to create, store, display and share custom sequences.

The main idea behind this app was to allow people to add a display of this type in a very simple way, reusing an old smartphone. It won't be as comfortable as dedicated electronics, but it allows for greater customization.

The app is (and will be) free. There are no premium features, no ads, no data gathering and it will not require any strange permission. If you want, you can invite me to a coffee, which will help to keep me awake to continue developing projects like this and to keep sharing them for free.
